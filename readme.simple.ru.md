# Attention Rollout для трейдинга - Простое руководство

## Что такое Attention Rollout?

Представьте, что вы детектив, пытающийся понять, почему торговый ИИ принял конкретное решение. Attention Rollout - это как "трекер мыслей", который показывает, на что именно "смотрел" ИИ, когда решил купить или продать.

Подумайте об этом так: когда вы решаете купить акцию, вы можете думать о вчерашней цене, тренде за неделю и каких-то новостях. Attention Rollout показывает версию этого мыслительного процесса у ИИ.

```
Ваш мозг при торговле:
"Хм, цена упала вчера... но она росла всю неделю..."
     [Вчера: 30%] [Эта неделя: 50%] [Новости: 20%]
                           ↓
                    Решение: КУПИТЬ

Мозг ИИ (показанный через Attention Rollout):
"Обрабатываю исторические данные..."
     [День -5: 5%] [День -4: 8%] [День -3: 12%] [День -2: 25%] [День -1: 50%]
                                      ↓
                               Сигнал: КУПИТЬ
```

## Почему это важно для трейдеров?

### Проблема "чёрного ящика"

Большинство торговых систем на ИИ - это "чёрные ящики" - они дают прогноз, но вы понятия не имеете почему:

```
Традиционный ИИ:
┌─────────────────┐
│    ВХОД         │
│  Данные цен     │──────────?????????────────▶  КУПИТЬ!
│  Объём и т.д.   │
└─────────────────┘
      Что происходит внутри? Никто не знает!
```

### Attention Rollout открывает ящик

```
С Attention Rollout:
┌─────────────────┐
│    ВХОД         │    "Я смотрю в основном на
│  Данные цен     │──▶  последние 2 дня, потому  ──▶  КУПИТЬ!
│  Объём и т.д.   │     что там явный восходящий тренд"
└─────────────────┘
      Теперь вы видите логику ИИ!
```

## Аналогия с рестораном

Представьте, что вы просите друга выбрать ресторан:

**Без Attention Rollout:**
Друг: "Пойдём в итальянский Mario's!"
Вы: "Почему?"
Друг: "Не знаю, просто так чувствую."

**С Attention Rollout:**
Друг: "Пойдём в итальянский Mario's!"
Вы: "Почему?"
Друг: "Ну, я подумал о рейтинге (40% моего решения), о расстоянии (30%), и о том, что ты упоминал, что любишь пасту на прошлой неделе (30%)."

Теперь вы понимаете их логику и можете больше доверять решению!

## Основная идея

Модели трансформеров (ИИ, стоящий за современными торговыми системами) работают, "обращая внимание" на разные части входных данных. Но у них несколько слоёв, и каждый слой смотрит на вещи по-разному.

```
Слой 1: "Смотрю на всё одинаково"
   [День-5] [День-4] [День-3] [День-2] [День-1]
     20%     20%      20%      20%      20%

Слой 2: "Начинаю фокусироваться на недавних днях"
   [День-5] [День-4] [День-3] [День-2] [День-1]
     10%     15%      20%      25%      30%

Слой 3: "Сильно фокусируюсь на вчерашнем дне"
   [День-5] [День-4] [День-3] [День-2] [День-1]
      5%     10%      15%      25%      45%

Attention Rollout объединяет все слои и показывает:
"В целом, вчерашний день имел наибольшее значение для этого прогноза"
```

## Простой торговый пример

Допустим, у вас есть ИИ, прогнозирующий цены Bitcoin:

### Входные данные (последние 5 дней)
```
День -5: $40,000 (небольшое падение)
День -4: $39,500 (продолжение падения)
День -3: $38,000 (сильное падение)
День -2: $39,000 (начало восстановления)
День -1: $41,000 (сильное восстановление)
```

### Прогноз ИИ: КУПИТЬ

### Attention Rollout показывает:
```
На что фокусировался ИИ:

День -5: ██               8%
День -4: ███             12%
День -3: ████            15%
День -2: ██████████      35%  ← Начало восстановления!
День -1: ████████        30%  ← Сильное восстановление!

ИИ научился, что паттерны восстановления важнее всего!
```

## Три типа паттернов внимания

### 1. Паттерн импульса (Momentum)
ИИ фокусируется на недавних данных - он следует тренду.

```
[Старое] [...]  [...] [Недавнее] [Последнее]
   5%     10%   15%      30%        40%

Интерпретация: "Текущий тренд продолжится"
Подходит для: Трендовых рынков
```

### 2. Паттерн возврата к среднему
ИИ смотрит на исторические средние - ожидает возврата цен к норме.

```
[Старое] [...]  [...] [Недавнее] [Последнее]
  25%     20%   20%      20%        15%

Интерпретация: "Текущая цена аномальна, вернётся к норме"
Подходит для: Боковых рынков
```

### 3. Паттерн фокуса на событии
ИИ концентрируется на конкретных важных моментах.

```
[Старое] [СОБЫТИЕ] [...] [Недавнее] [Последнее]
   5%      60%     10%      15%        10%

Интерпретация: "То конкретное событие - ключевое"
Подходит для: Торговли на новостях
```

## Как Attention Rollout помогает в торговле

### 1. Валидация решений модели

Прежде чем доверять сигналу КУПИТЬ, проверьте, на что модель обратила внимание:

```
Хороший знак:
- Модель фокусировалась на релевантных технических паттернах
- Внимание разумно распределено по важным периодам
- Недавнее ценовое движение имеет значительный вес

Тревожный знак:
- Модель фокусировалась на нерелевантных старых данных
- Внимание сконцентрировано на одной случайной точке
- Недавние важные события проигнорированы
```

### 2. Понимание рыночных режимов

```
Внимание на бычьем рынке:    Внимание на медвежьем рынке:
[....] [....] [НЕДАВНЕЕ]     [ИСТОРИЯ] [....] [....]
         ↓                        ↓
  Импульсная торговля        Осторожность, взгляд назад
```

### 3. Отладка плохой производительности

Когда модель теряет деньги, attention rollout показывает почему:

```
Анализ убыточной сделки:
"Модель смотрела не на то!"

Ожидалось: Фокус на вчерашнем прорыве
Фактически: Фокус на данных двухнедельной давности

Решение: Переобучить с акцентом на недавние паттерны
```

## Распространённые ошибки

### Ошибка 1: Игнорирование энтропии внимания

Энтропия измеряет, насколько "распределено" внимание.

```
Низкая энтропия (концентрированно):    Высокая энтропия (распределено):
[█] [█] [█] [█] [████████]             [██] [██] [██] [██] [██]

Концентрация может означать:           Распределение может означать:
- Чёткий сигнал                        - Неопределённость
- Или переобучение!                    - Или комплексный анализ
```

### Ошибка 2: Не сравнивать прибыльные и убыточные сделки

Всегда анализируйте оба типа!

```
Внимание прибыльных сделок:      Внимание убыточных сделок:
Фокус на недавнем импульсе       Фокус на старых, нерелевантных данных

Этот инсайт помогает улучшить модель!
```

### Ошибка 3: Забывать о множественных головах

Модели трансформеров имеют несколько "голов внимания" - как несколько экспертов. Некоторые головы могут ошибаться!

```
Голова 1: Смотрит на объём        → Полезно
Голова 2: Смотрит на старые цены  → Не полезно
Голова 3: Смотрит на импульс      → Полезно
Голова 4: Смотрит на шум          → Вредно

Attention Rollout усредняет их, но вы также можете
анализировать каждую голову отдельно для глубокого анализа.
```

### Ошибка 4: Доверять без проверки

Внимание показывает корреляцию, а не причинность!

```
Модель фокусируется на данных понедельника → Предсказывает цену вторника
                   ↓
Понедельник действительно предсказывает вторник?
Или модель учит ложные паттерны?

Всегда проводите бэктест для проверки!
```

## Быстрый пример на Python

```python
# Простой пример использования attention rollout
import torch
from attention_rollout import AttentionRollout
from model import TradingTransformer

# Создаём модель и анализатор rollout
model = TradingTransformer(input_dim=5, d_model=64, n_heads=4, n_layers=2)
rollout = AttentionRollout(model)

# Ваши ценовые данные (20 дней: open, high, low, close, volume)
price_data = torch.randn(1, 20, 5)  # На практике используйте реальные данные!

# Получаем прогноз и видим, на что модель обратила внимание
prediction, _ = model(price_data)
attention_scores = rollout.get_input_attribution(price_data)

# Выводим, какие дни были важнее
print("Важность дня для прогноза:")
for i, score in enumerate(attention_scores):
    bar = "█" * int(score * 50)
    print(f"День {-20+i}: {bar} {score:.2%}")
```

Вывод может выглядеть так:
```
Важность дня для прогноза:
День -20: █ 2%
День -19: █ 3%
...
День -2: █████████████ 26%
День -1: ██████████████████ 35%
```

## Ключевые выводы

1. **Attention Rollout** показывает, на что фокусируется ваш торговый ИИ
2. **Интерпретируемость** помогает доверять (или не доверять) прогнозам модели
3. **Разные паттерны** указывают на разные рыночные стратегии
4. **Сравнение сделок** выявляет, что работает, а что нет
5. **Всегда проверяйте** - внимание показывает корреляцию, а не причинность

## Итог

Attention Rollout превращает ваш торговый ИИ из "чёрного ящика" в прозрачного партнёра. Вместо слепого следования сигналам, вы можете понять, почему модель делает каждый прогноз, и принимать лучшие торговые решения.
